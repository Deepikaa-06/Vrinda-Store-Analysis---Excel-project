# 📊 Vrinda Store Data Analysis (Excel Project)

## 📌 Project Overview

This project focuses on analyzing sales data of **Vrinda Store** using Microsoft Excel. The goal is to extract meaningful insights from raw data and present them through an interactive dashboard.

---

## 🎯 Objective

* Analyze sales performance of the store
* Identify top-performing products and categories
* Understand customer behavior
* Track revenue trends over time
* Provide business insights for better decision-making

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Power Query (for data cleaning)
* Data Visualization techniques

---

## 📂 Dataset Description

The dataset contains information such as:

* Order ID
* Order Date
* Customer details
* Product category
* Sales amount
* Quantity
* Region/Location

---

## 🧹 Data Cleaning Steps

* Removed duplicate records
* Handled missing values
* Standardized column formats
* Converted data types where required
* Cleaned inconsistent entries using Power Query

---

## 📊 Dashboard Features

The Excel dashboard includes:

* 📈 Sales trend over time
* 🛍️ Top-selling products
* 🌍 Region-wise sales analysis
* 👥 Customer segmentation
* 💰 Revenue distribution

---

## 🔍 Key Insights

* Identified best-performing product categories
* Found peak sales periods
* Observed customer purchase patterns
* Highlighted regions contributing highest revenue


## 🙌 Conclusion

This project demonstrates how Excel can be used as a powerful tool for data analysis and visualization. It helps in transforming raw data into actionable business insights.
